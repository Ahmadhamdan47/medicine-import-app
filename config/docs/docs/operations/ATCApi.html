atcAddPost

Add/Post New ATC Code.

URL

http://localhost:3000/atc/add

HTTP Method

POST

Response Type

Parameters

atcAllGet

Retrieve all ATC Codes.

URL

http://localhost:3000/atc/all

HTTP Method

GET

Response Type

Parameters

atcDrugIDGet

Retrieve ATC code associated with a drug ID.

URL

http://localhost:3000/atc/{drugID}

HTTP Method

GET

Response Type

Parameters

  • Path: drugID Integer

    The ID of the drug to retrieve the ATC code for.

atcIdDelete

Delete an ATC code by ID.

URL

http://localhost:3000/atc/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id Integer

    ID of the ATC code to delete.
